# ZweToe Pharmacy Ecommerce and Management System (Full-Stack)

A modern Pharmacy POS and Inventory Management System built with **Laravel (API)** and **Next.js (Frontend)**.

## 🚀 Project Overview

This project is being developed module-by-module to ensure a robust and scalable architecture. It handles medicine inventory, sales transactions, and expiry alerts.
